# Iphone-purchase-Prediction-using-Machine-Learning
This project predicts whether a customer will purchase an iPhone based on various features using machine learning algorithms.

## Libraries Used
- 'pandas' : Data handling
- 'matplotlib', 'seaborn' :  Data visualization  
- 'scikit-learn' : Machine Learning models and evaluation
- 'joblib' : Saving trained models

## Algorithms Implemented
- K-Nearest Neighbors (KNN)  
- Logistic Regression  
- Linear Regression  
- Decision Tree Classifier  
- Random Forest Classifier

## Features
- Data exploration and visualization  
- Splitting dataset into train and test sets  
- Training multiple machine learning models  
- Model evaluation using accuracy, confusion matrix, classification report, and precision score  
- Saving trained models with joblib

## Dataset
The dataset contains customer information and purchase history.

## How to run
1. Install required libraries  
2. Run the notebook or script  
3. Train models and predict iPhone purchase outcomes
